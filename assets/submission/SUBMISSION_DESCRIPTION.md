# DELPHAI

## Tagline

An AI reasoning council that checks the math, explains the risk, and refuses to burn out your team.

## Keywords

Microsoft Foundry, Reasoning Agents, Foundry IQ, Multi-Agent System, Agent Framework, Azure AI Search, Microsoft Learn MCP, Responsible AI, Certification Readiness, Enterprise Learning, Synthetic Data, Agent Orchestration, Evaluations, Telemetry

## Description

DELPHAI is a multi-agent enterprise learning system for certification readiness. A council of specialized agents reasons over team skills, exam-domain gaps, workload capacity, grounded learning content, and historical synthetic outcomes to decide whether a team can realistically meet a certification deadline.

Unlike a simple study-plan chatbot, DELPHAI decomposes the decision across agents: a curator grounds learning paths in approved sources, an assessment agent checks readiness per exam domain, a planner proposes the optimistic path, a skeptic stress-tests the math, a wellbeing agent refuses unsustainable overwork, and a conductor reconciles the debate into a GO / NEGOTIATE / NO-GO verdict.

The app uses Microsoft Foundry and Microsoft Agent Framework paths, Foundry IQ-style grounding through Azure AI Search over synthetic knowledge documents, Microsoft Learn MCP for official learning references, and deterministic code-backed reasoning for pass probability, capacity, coverage gates, and Poisson-binomial team forecasts. It includes synthetic-only data, citations, telemetry, automated evaluations, and a live demo UI designed for a clear hackathon presentation.
